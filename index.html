Sobre Mim

Olá! Meu nome é André Lima, sou um Engenheiro de Software com foco em desenvolvimento Full-Stack. Minha paixão é utilizar a tecnologia para construir soluções eficientes, seja para a web ou para desktop, aplicando sempre as melhores práticas de programação e metodologias ágeis como o Scrum.

Trabalho com tecnologias modernas como JavaScript, TypeScript, React e Node.js para criar aplicações robustas e escaláveis. Minha experiência abrange desde a gestão de bancos de dados relacionais como PostgreSQL até NoSQL como MongoDB, garantindo a escolha da melhor tecnologia para cada desafio.

Atualmente, estou imerso em um projeto de residência em software pela UEMA em parceria com a BRISA, onde colaboro com uma equipe talentosa para resolver desafios complexos do mundo real. Estou sempre buscando aprender, evoluir e contribuir para projetos inovadores. Vamos construir algo incrível juntos?
